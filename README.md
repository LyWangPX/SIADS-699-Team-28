# SIADS-699-Team-28

This repository contains the capstone project work for Team 28. The project is divided into two distinct modules, each representing individual contributions to the analysis of stock market data.

**Please Note:** The code for each module operates independently and should be run and viewed separately.

## Team Members & Modules

### Daniel Lagalante
**Focus:** Traditional Stock Indicator Analysis
- **Repository:** [ChartPatterns](https://github.com/djl6zjhb/ChartPatterns) (linked as submodule)
- This module focuses on the analysis of traditional technical indicators and chart patterns for stock market prediction.

### Yifan Wang
**Focus:** Stock Latent Space Analysis
**Repository:** [capstone-umich-yifan](https://github.com/LyWangPX/capstone-umich-yifan) (linked as submodule)
- This module applies deep learning techniques to identify latent market regimes and patterns using autoencoders and clustering algorithms.

## Getting Started

To clone this repository including the submodules, use:

```bash
git clone --recursive https://github.com/LyWangPX/SIADS-699-Team-28.git
```

If you have already cloned the repository without submodules, run:

```bash
git submodule update --init --recursive
```
